## db-designer 主要功能

* 可视化设计数据库表结构
* 元数据管理
* 版本管理
* 数据库比对与同步
* AI 自动设计 / 修改表结构 / 推荐索引
* 通过 SQLite 本地存储项目数据与配置
* 通过 Git 管理设计数据（适配 GitHub / GitLab / Gitee）
* 编程对象维护（函数 / 存储过程 / 触发器，支持对比、同步、导出）
* SQL 导出（表结构、索引、元数据、编程对象）

## 详细功能

### 设计数据库

* 支持 MySQL / PostgreSQL / Oracle
* 可视化维护表、字段、索引，支持字段拖拽排序
* 支持自定义常用字段
* 支持内置类型 + 自定义类型，快捷选择
* 支持单表 SQL 预览与项目整体 SQL 导出

### 元数据管理

* 为表维护初始化数据 / 元数据
* 支持单表 Excel / CSV 导入
* 支持 INSERT 语句导出

### 版本管理

* 从当前项目状态创建版本快照（包含表结构、索引、元数据、编程对象）
* 通过选择历史版本导出升级脚本
* 导出当前版本整体 SQL
* 支持删除版本

### 数据库比对与同步

* 支持在设置页统一维护数据库连接
* 支持对比本地模型与远程数据库结构差异
* 支持生成增量同步脚本
* 支持将远程表结构与编程对象同步回本地模型

### 编程对象管理

* 支持函数、存储过程、触发器的增删改查
* 支持按数据库类型筛选导出
* 支持远程对比与一键同步

### AI 能力

* 支持配置兼容 OpenAI API 格式的模型服务
* 通过自然语言生成项目支持的表结构格式并一键落库
* AI 设计时自动注入当前项目上下文（已有表、索引、元数据）
* 支持 AI 修改已有表结构
* 支持基于业务场景和慢查询推荐索引
* 支持保存长期复用的 AI 设计通用提示词
* AI评审的功能，检测潜在的设计问题并给出建议

### Git 数据同步

* 支持 Git 仓库初始化、信息查看、推送同步
* 支持从远程仓库拉取并覆盖本地数据（带危险操作确认）
* 支持保存 Git 平台、令牌、仓库等配置

### 本地配置与应用能力

* 所有数据基于 SQLite 本地存储，无需服务端
* 支持本地持久化 AI 配置、Git 配置、自定义类型等设置
* 启动应用后自动检查更新
