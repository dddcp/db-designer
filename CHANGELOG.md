## [0.4.3] - 2026-03-21

### 新增
- 使用 OpenSpec 工作流管理功能开发（skills、prompts、specs）
- 编程对象支持数据库类型筛选导出
- 编程对象同步时支持按数据库类型筛选
- SQL 导出支持包含编程对象（函数、存储过程、触发器）
- 升级脚本支持编程对象变更（新增/删除/修改）

### 优化
- 编程对象模块代码重构

## [0.4.2] - 2026-03-19

### 新增
- 打开应用后自动检查更新，发现新版本时显示更新按钮
- Git配置页面新增"清除配置"功能，可清除Git平台、令牌和仓库设置
- 首页同步按钮根据Git配置状态显示/隐藏，未配置Git时隐藏同步按钮
- 批量同步表结构

### 修复
- 导出sql中去掉表的中文说明
- 修复中文命中有换行符的问题

## [0.4.1] - 2026-03-15

### 新增
- 编程对象（函数、存储过程、触发器）管理模块：支持增删改查、远程对比、一键同步、SQL 导出（独立模块，不支持多语言）
- 数据库同步新增编程对象对比 Tab，支持查看本地与远程差异并同步
- 在表列表中添加搜索功能，增强用户体验
- 适配Oracle

### 修复
- 针对默认值和 NULL 的处理优化
- 复合主键校验、非空数值字段校验
