# 更新日志

所有版本的变更记录。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。

## [0.4.1] - 2026-03-15

### 新增
- 编程对象（函数、存储过程、触发器）管理模块：支持增删改查、远程对比、一键同步、SQL 导出（独立模块，不支持多语言）
- 数据库同步新增编程对象对比 Tab，支持查看本地与远程差异并同步
- 在表列表中添加搜索功能，增强用户体验
- 适配Oracle

### 修复
- 针对默认值和 NULL 的处理优化
- 复合主键校验、非空数值字段校验
